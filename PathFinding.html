Dijkstra's Algorithm Grid Visualizer

Instructions:

  • Click Set Start Node and then click on a cell to place the start node.
  • Click Set End Node and then click on a cell to place the end node.
  • Click Draw Obstacles and then click on cells to draw obstacles.
  • Click Find Path to visualize the shortest path from start to end node.
  • Click Reset Grid to clear the grid and start over.
